## Typography & Scale

### Primary Typeface
**JetBrains Mono** (or closest mono fallback)

### Hierarchy
- **HEADLINE:** ALL CAPS, heavy weight
- **LABELS:** [ BRACKETED ], mustard
- **BODY:** short, white, functional

### Never use
- Script fonts
- Soft rounded fonts
- Mixed typefaces on one sign

---

## Colour Discipline

| Colour | Usage | Hex |
|--------|-------|-----|
| White | Information | #FFFFFF |
| Mustard | Indicators / labels only | #E3B04B |
| Amber | Actions / booking | #FF9F1C |
| Green tint | Background or panel field | Subtle |
| Red | Unavailable / offline ONLY | #FF0000 |

### Never use
- Gradients
- Photos behind text
- Soft shadows
- Decorative elements
